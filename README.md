Many R packages support Venn diagram.

- `VennDiagram`
- `colorfulVennPlot`
- `venn`
- `eulerr`
- `venneuler`
- `gplots`
- `ggvenn`

and `ggVennDiagram`.

Why choose `ggVennDiagram`?

This book is originally for notes in the development of `ggVennDiagram`.
And it may be used as a Venn plot gallery in R environment.
